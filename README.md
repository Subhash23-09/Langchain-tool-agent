### Architecture Description

This project implements a **LangChain agent integrated with a Gradio web interface** to process user queries and generate intelligent responses using both tools and a large language model.

1. **User Input (Gradio)**
   The user enters a query through the Gradio text input interface. The interface provides a simple chat-style interaction.

2. **`chat()` Function**
   The `chat()` function receives the user input and acts as the controller. It forwards the input to the LangChain agent for processing and manages the UI-only chat history.

3. **LangChain Agent**
   The LangChain agent serves as the decision-making component of the system. Based on the userâ€™s input, it automatically determines whether to:

   * Invoke an advanced tool (such as `add_numbers`, `multiply_numbers`, `positive_prompt_tool`, `negative_prompt_tool`, or `suicide_related_tool`), or
   * Generate a direct response using the **Gemini 2.5 Flash** large language model.

   The agent abstracts tool selection logic and ensures the appropriate function is executed without manual intervention.

4. **Conversation Memory**
   The system uses `ConversationBufferWindowMemory` to store recent conversation context. This enables coherent multi-turn interactions.
5. **Output to Gradio**
   The final response generated by the agent is returned to the Gradio interface and displayed to the user in a chat format.

### Libraries Used

| Library                  | Purpose                                                                                                    |
| ------------------------ | ---------------------------------------------------------------------------------------------------------- |
| `langchain_google_genai` | Provides the Gemini LLM wrapper (`ChatGoogleGenerativeAI`) for generating AI responses.                    |
| `langchain.agents`       | Implements `Tool`, `initialize_agent`, and `AgentType` for automatic tool routing and agent orchestration. |
| `langchain.memory`       | Provides `ConversationBufferWindowMemory` to maintain conversation context for multi-turn chats.           |
| `gradio`                 | Creates a web-based interface for user interaction (text input/output).                                    |
| `re`                     | Extracts numbers from user input using regular expressions.                                                |
| `os`                     | Handles environment variables (e.g., storing the Google API key).                                          |
